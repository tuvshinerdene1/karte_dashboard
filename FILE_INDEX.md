## 📊 Features at a Glance

✅ **4-Stage Pipeline Animation**
```
цаг захиалга → эмч → ЭМД → DONE
```

✅ **Real-time Bottleneck Detection**

- Yellow (33-66%) = Warning  
- Red (66-100%) = Critical

1. Process Flow (main animation)
2. Alerts (alerts & warnings)

- Orbital particles
- Flowing connectors
